# JobPilot AI 交互状态设计

更新时间：2026-06-24

## 1. 状态设计原则

JobPilot AI 有大量异步和 AI 生成流程，状态设计必须明确。用户要知道系统正在做什么、做到了哪一步、失败后如何继续。

每个核心流程都要覆盖：

- 初始状态
- 加载状态
- 成功状态
- 空状态
- 失败状态
- 可重试状态
- 权限状态

## 2. 简历解析状态

### 2.1 上传中

文案：

```text
正在上传简历...
```

表现：

- Upload 组件显示进度
- 禁用重复上传

### 2.2 解析中

文案：

```text
正在解析简历文本...
```

辅助说明：

```text
当前版本支持文本型 PDF 和 DOCX，不支持扫描版 PDF 或图片简历。
```

### 2.3 解析失败

文案：

```text
没有识别到可用文本，你可以改为粘贴文本简历。
```

操作：

- 重新上传
- 粘贴文本

## 3. 作战卡生成状态

### 3.1 生成前确认

确认信息：

- 当前岗位
- 当前简历版本
- 行业领域
- 岗位方向

操作：

- 确认生成
- 更换简历

### 3.2 缓存命中

文案：

```text
已复用相同简历和岗位 JD 的分析结果。
```

表现：

- 在作战卡顶部展示“缓存结果”标签
- 允许重新生成

### 3.3 AI 生成中

阶段提示：

```text
正在分析岗位 JD
正在对比简历经历
正在整理岗位作战卡
```

设计要求：

- 不使用无限空白等待
- 显示步骤进度
- 允许用户离开页面后稍后回来查看

### 3.4 AI 返回格式错误

文案：

```text
AI 返回结果格式异常，暂时无法生成作战卡。
```

操作：

- 重试
- 查看原始结果
- 返回岗位详情

## 4. 题目生成状态

### 4.1 检索公共题库

文案：

```text
正在从公共题库检索匹配题目...
```

展示指标：

- 已提取标签数
- 已召回题目数
- 高质量题目数

### 4.2 AI 补题

文案：

```text
公共题库覆盖不足，正在补充生成缺口题目...
```

展示缺口：

- 缺少的能力标签
- 缺少的题目类型
- 目标补充数量

### 4.3 候选题入库

文案：

```text
AI 补充题目已进入候选题库，审核通过后可沉淀为公共题库。
```

状态：

- 待审核
- 已通过
- 已拒绝
- 相似重复

## 5. 答案评分状态

### 5.1 答案为空

文案：

```text
先写下你的回答，再提交 AI 评分。
```

### 5.2 评分中

文案：

```text
正在从岗位要求、参考答案和你的回答三个角度评分...
```

### 5.3 评分成功

展示顺序：

1. 总分
2. 回答优点
3. 主要问题
4. 优化后的回答
5. 面试官可能追问

### 5.4 评分失败

文案：

```text
评分失败，请稍后重试。你的原始回答已保留。
```

操作：

- 重试评分
- 保存草稿

## 6. 知识库保存状态

### 6.1 保存前

用户可编辑：

- 标题
- 分类
- 标签
- 内容
- 掌握程度

### 6.2 保存成功

文案：

```text
已保存到个人知识库。
```

操作：

- 查看知识条目
- 继续练习

### 6.3 重复保存

文案：

```text
已存在相似知识条目，你可以更新原条目或另存为新条目。
```

## 7. 投递复盘状态

### 7.1 无结果

文案：

```text
先记录投递进度，拿到结果后再生成复盘建议。
```

### 7.2 可复盘

触发条件：

- 有面试结果
- 有失败原因或反馈
- 有关联岗位和作战卡

### 7.3 复盘生成中

文案：

```text
正在结合岗位要求、练习记录和投递结果生成复盘建议...
```

## 8. 权限与登录状态

### 8.1 未登录

行为：

- 业务接口返回 401
- 前端跳转 `/login`
- 登录后回到原页面

### 8.2 无权限

文案：

```text
你没有权限访问该内容。
```

说明：

- 用户只能访问自己的简历、岗位、作战卡、练习、知识库和投递记录。

## 9. 限流与成本提示

AI 生成触发限流时：

```text
AI 请求过于频繁，请稍后再试。
```

建议在按钮附近显示：

- 今日可用次数
- 本次是否可能调用 AI
- 是否命中缓存

## 10. 全局错误策略

错误提示应包含：

- 发生了什么
- 用户能做什么
- 原输入是否已保存

避免只展示：

```text
系统错误
```

推荐：

```text
生成失败，岗位和简历内容已保存。你可以稍后重试。
```
